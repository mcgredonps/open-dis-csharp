The files contained in C#->DIS#1.0 are the original source generated with open-dis_3.1.3.jar.  Newer version will be placed in C#->DIS#2.0.  The newer version will have changes that will break any source making use of DIS #1.0.  These changes will be in Method names and Namespaces.

A copy of the XMLPG folder that was used to generate the DIS#1.0 source is also contained in this directory.
